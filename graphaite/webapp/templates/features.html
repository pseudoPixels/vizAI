Automated Visualitions using AI

Graphaite uses AI and Machine Learning to automate and accelerate the visualizations to find important insights from dataset. After connecting the dataset, we can also set the feature and class variable for automatic data visualizations, with minimal manual effort. Currently, in this prototype version of Graphaite, it only supports .csv formatted dataset.

Automated Dashboard Development

Graphaite also automatically creates Dashboard comprising of the auto generated visualizations. The individual visualizations are categorized as per the feature variables for easier management and presentation.

Interactive Graphs

The graphs generated are interactive (i.e., using Plotly in this prototype version). In contrast to static (i.e., png/jpg) image, you can interact with the visualizations for further investigation of the plotted data points, such as, zoom-in, zoom-out, selecting subset of the data points, investigating the correponding data values and so on.

Customize Auto Generated Visualitions

You can take the auto generated visualizations as base for further customization as per your likings. This could be an efficient worflow to accelerate the insight finding and reporting via visualizations, as the majority of the heavy lifting (i.e., searching for insight and building visualizations) are already done using AI and Machine Learning.
