# vizAI

### Installation
```buildoutcfg
>>> git clone https://github.com/pseudoPixels/vizAI.git
>>> cd vizAI
>>> conda create -n vizAiEnv python=3.6
>>> conda activate vizAiEnv
>>> pip install -e .
```

### App Start
```buildoutcfg
>>> sh run.sh
```
